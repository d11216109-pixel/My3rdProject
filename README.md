# My3rdProject - 猜數字遊戲

## 專案說明
這是一個用 Python 實作的猜數字遊戲。

## 遊戲規則
1. 系統會隨機產生一個 1 到 100 之間的整數答案
2. 玩家輸入猜測的數字
3. 系統會提示「大了」或「小了」來引導玩家
4. 猜對時顯示「恭喜通過！」並顯示總共猜測的次數
5. 遊戲記錄每次的猜測次數，可以看到自己的進步

## 如何執行
```bash
python3 GN.PY
```

## 遊戲特色
- 輸入驗證：確保玩家輸入的是有效的數字
- 範圍檢查：提示玩家輸入 1 到 100 之間的數字
- 計次功能：記錄並顯示總共猜測了幾次
- 友善提示：清楚的中文提示訊息

## 範例遊玩過程
```
請猜一個 1 到 100 之間的數字: 50
大了
請猜一個 1 到 100 之間的數字: 25
小了
請猜一個 1 到 100 之間的數字: 37
恭喜通過！你總共猜了 3 次。
```
